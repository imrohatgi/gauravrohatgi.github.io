# Images Directory

This folder should contain the following images from your website:

## Required Images:

1. **profile.jpg** - Your professional profile photo for the homepage hero section
2. **gold-titan-award.jpg** - Gold Titan Award 2024 certificate/badge image
3. **ieee-logo.png** - IEEE organization logo
4. **acm-logo.png** - ACM organization logo

## How to Add Images:

1. Download images from your existing website at https://www.gauravrohatgi.com/
2. Save them in this `images/` folder
3. Uncomment the image tags in the HTML files:
   - `index.html` - Profile photo
   - `awards.html` - Gold Titan Award image
   - `memberships.html` - IEEE and ACM logos

## Optional Images:

- Certification badges from LinkedIn Learning
- Screenshots of media coverage
- Research paper thumbnails
- Additional professional photos

## Image Format Recommendations:

- Profile photo: 500x500px or similar square aspect ratio, JPG format
- Logos: PNG with transparent background, 200-300px width
- Awards: JPG or PNG, 400-600px width
- Keep file sizes under 500KB for faster loading
